# Chasium

![](./docs/images/WhiteBg.png)

一个TRPG多人游戏工具。

Chasium是一个植物学词根，意为“聚伞花序”。选其作为项目名的原因是它的样子与**Call of Cthulhu** TRPG的发行商Chaosium非常相似。项目Logo即为植物学中聚伞花序的花图式。
